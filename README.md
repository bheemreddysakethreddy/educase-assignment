# Educase Login / Signup Assignment

This project is a responsive Login and Signup UI built using **React + Vite + TailwindCSS**.

## Live Demo

[https://educase-assignment-three-lilac.vercel.app/]

## Tech Stack

* React
* Vite
* TailwindCSS
* React Icons

## Project Structure

src/
pages/
ui/
App.jsx

## Installation

Clone the repository:

git clone https://github.com/bheemreddysakethreddy/educase-assignment.git

Install dependencies:

npm install

Run the development server:

npm run dev

## Features

* mobile first UI
* Login form
* Signup form
* Account settings page
* Profile avatar with camera icon
